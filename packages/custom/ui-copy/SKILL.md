# UI Copy

- 文案站在使用者视角，不替系统说话。
- 删除自我解释、抽象空话、提示词残留。
- 没信息价值的大标题不要写。
- 冗余的小标题、眉标题、分组标题能删就删。
- 说明文字尽量贴近操作，不要另起空块。
- 需要示例时读 `references/copy-patterns.md`
