# UI Ext

- 小窗优先级极高：主任务、当前状态、主操作先出来。
- 控制高度和层级，不要塞成小后台。
- 按钮反馈必须明显，不然用户会重复点。
- popup 里最容易出现冗余标题、空耗空间和同权重按钮。
- 细化规则：`references/platform.md`
- 起手 brief：`../ui-assets/references/platform-briefs.md`
