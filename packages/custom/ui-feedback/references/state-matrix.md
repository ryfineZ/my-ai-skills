# State Matrix

## Primary Button

- 按下：`<=100ms` 内有压感或高亮变化
- 异步：`>300ms` 必须进入 `loading`
- 成功：原位确认、状态文案或轻提示
- 失败：错误贴近触发点，并给重试或修正入口

## Destructive Action

- 默认弱于主按钮
- 执行前确认风险
- 执行后给撤销或结果提示

## Form Submit

- 不可提交时明确禁用原因
- 提交中禁重复点
- 失败后聚焦到问题字段或问题区域

## Toggle / Switch

- 切换结果要即时可见
- 异步保存时要告知“正在保存”或“已保存”

## List / Table Row Action

- 行内操作只留最常用的
- 多余动作收进更多菜单
- 同层不要摆一排同权重按钮
