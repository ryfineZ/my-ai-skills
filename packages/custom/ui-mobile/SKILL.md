# UI Mobile

- 先保主任务，再放次信息。
- 触控热区、按钮反馈、滚动节奏优先。
- 不要把桌面布局硬压成窄屏版。
- 小屏更容易暴露冗余标题、冗余按钮和低对比度问题。
