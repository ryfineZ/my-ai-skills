# Template Sources

## Direct Template Sources

- `Page UI`: 适合 Web 官网、landing page、marketing page。

## Block / Pattern Sources

- `Magic UI`: 适合 hero、CTA、showcase、logo rail、轻动效展示区。
- `shadcn/ui`: 适合 dashboard、settings、form、card、table 等页面骨架与组件组合。
- `Vant`: 适合 H5 / mobile web 的区块、表单、列表和移动布局模式。
- `Vant Weapp`: 适合小程序列表页、设置页、表单页、详情页。
- `TDesign Miniprogram`: 适合企业、工具、后台感更强的小程序页面。
- `WeUI`: 适合保守、官方、低风险的 H5 / 微信生态界面。

## Shell / Boilerplate Only

- `Plasmo`
- `WXT`
- `Electron React Boilerplate`
- `Expo`

这些项目主要用来搭工程，不直接作为视觉模板。它们负责项目结构和运行环境，界面风格应从上面的模板源或模式源迁移。
