# Style Directions

## Core

- 先选一个主方向，不要混搭太多风格。
- 先做结构和节奏，再做装饰。
- “更高级”通常来自克制、统一和对比，而不是更多特效。

## Reliable Directions

### Tooling

- 关键词：专业、克制、稳定、耐看
- 适合：扩展、桌面工具、管理台、开发者产品

### Editorial

- 关键词：留白、排版、内容权重、秩序感
- 适合：文档、阅读、内容产品、品牌页面

### Data Panel

- 关键词：信息层级、对比、节奏、状态感
- 适合：仪表盘、监控、分析、工作台

### Brand Display

- 关键词：氛围、视觉记忆点、首屏叙事、克制动效
- 适合：官网、产品页、展示页

### Tahoe Glass

- 关键词：液态玻璃、通透、晶莹、分层、柔和反射、轻盈、高级
- 适合：macOS 风格桌面应用、工具类控制面板、偏 Apple 气质的设置页和侧边栏
- 材质：半透明玻璃层、柔和高光、轻微折射、边缘发亮，但不能糊成一片
- 层级：用透明度和模糊制造深度，主内容仍然必须最清晰，不能让玻璃材质抢走任务焦点
- 版面：避免营销页式巨幅 hero，首屏优先给任务、列表、筛选和当前状态
- 字体：中文优先贴近 macOS 系统气质，避免过硬、过黑、过展示型的字重和字形
- 配色：默认走银灰、雾白、石墨、冷绿、浅青或暖光映射，不要习惯性落回蓝紫渐变
- 图标与控件：可以有微拟物和果冻感，但要克制，避免回到厚重 skeuomorphism
- 动效：轻、软、顺滑，强调光影和层次变化，不做弹跳炫技
- 个性化：适合搭配动态背景或轻微色彩映射，但前提是文字与控件可读
- 使用边界：优先用于桌面端或偏系统风产品；移动端、小程序、信息密集后台默认不要硬套
- 风险提醒：
  - 最外层窗壳只是承载层，不要做得比内部内容层更实、更重
  - 不要把 Tahoe 直接做成被用烂的蓝紫玻璃模板
  - 不要把所有卡片都做成玻璃
  - 不要用过重模糊掩盖结构问题
  - 不要把导航、摘要和控制区做得比主任务区更抢眼
  - 不要写界面自我解释型文案，尤其不要讲“这个设计为什么高级”
  - 不要为了“高级感”增加无用外框/内框
  - 不要让背景干扰文字和图标可读性

## Anti-Patterns

- 紫色渐变白底的一般性 SaaS 模板风
- 没有品牌依据还习惯性回到蓝紫渐变、蓝紫玻璃或蓝紫主色
- 所有卡片同样重，所有按钮同样亮
- 没有留白，内容直接铺满
- 用原始 hex、阴影和圆角值到处硬写
- 一屏堆多个动效
- 为了“高级感”牺牲可读性和操作效率
- 把 Liquid Glass 做成一层糊玻璃蒙版，只有模糊没有结构
