# Blocking Checks

任一命中都算不通过：

- 主任务入口权重失衡
- 标题、卡片、按钮或输入区尺寸明显失衡
- 文案像系统在解释自己，或存在抽象空话、提示词残留、没信息价值的大标题
- 重复块、重复标题、冗余小标题、眉标题、分组标题没有删除
- 在没有明确品牌依据时，仍然默认落回蓝紫渐变、蓝紫玻璃、紫白 SaaS、银灰、雾绿等常见 AI 安全配色
- 配色没有明确主色关系，或一眼就是通用模板味
- 字体气质与产品类型不匹配
- 小号文字、辅助文字或玻璃层文字对比度过低，出现发虚、发灰、读不清
- 信息层级混乱，或间距、尺寸、容器明显空耗空间
- 嵌套了没有信息价值的外框、内框，容器层级过多
- 按钮点下后没有明确反馈，用户无法判断是否生效或是否处理中
- 同层按钮重复、近义、同权重，主次不清
- 状态缺失或像静态海报
- 为了“质感”牺牲可读性和操作效率
- 明显像 AI 默认模板拼出来的页面
